## 👤 `./about_me`

Soy un estudiante avanzado de **Ingeniería en Computación** con fuerte orientación hacia la **ciberseguridad defensiva**.
Mi pasión es entender cómo funcionan los ataques para poder defenderme de ellos mejor.

Me especializo en el lado azul: monitorización de eventos de seguridad, análisis de logs, correlación de alertas y respuesta a incidentes.
Creo en aprender haciendo — cada proyecto que construyo viene acompañado de documentación real.

- 🔵 **Blue Team** por convicción, no por descarte
- 🧪 Laboratorio personal con Kali Linux para práctica continua
- 📖 Formación activa en TryHackMe & HackTheBox
- 🤖 Automatización de tareas SOC con Python y Bash
